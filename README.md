# Gesture Virtual Keyboard

Gesture Virtual Keyboard is an AI/ML-based project that uses real-time hand gesture recognition to simulate keyboard input.  
By leveraging computer vision techniques and machine learning models, the system detects hand landmarks via a webcam and maps them to virtual keystrokes.
